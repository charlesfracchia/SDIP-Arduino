RE-START ARDUINO, then do:

Adam-Marblestones-MacBook-Pro:OBMPduino_demo1_v2 adammarblestone$ python send_time_to_arduino.py 
Which of these serial ports is the Arduino attached to?
0 : /dev/tty.Bluetooth-PDA-Sync
1 : /dev/tty.Bluetooth-Modem
2 : /dev/tty.MindWaveMobile-DevA
3 : /dev/tty.usbmodem411
Enter the number as 0 to N-1 where N is the number of serial ports: 3
/dev/tty.usbmodem411
